# Conecta Cidadão — Design System

**Slogan:** Você informa. A prefeitura resolve.

## Aesthetic Stance

**Civic Modern** — Clean, professional, trustworthy, and welcoming. Balances institutional credibility with active community feel. Not a social media feed, not old government portal, not aggressive complaint app. This is collaborative civic engagement.

## Typography

- **Display**: Manrope (headings, hero text, cards)
- **Body**: Inter (readable text, forms, descriptions)
- **Mono**: JetBrains Mono (status codes, technical labels when needed)

## Layout Principles

- **Mobile First**: Bottom navigation with 5 items, CTAs sized for easy tapping (44px min)
- **Generous Spacing**: Let content breathe, clear visual hierarchy
- **Border Radius**: 16-24px for cards
- **Shadows**: Subtle, modern depth
- **Icons**: Lucide React, linear style

## Component Style

- Cards with soft shadows and rounded corners
- Large, accessible buttons with clear states
- Color-coded category tags
- Smooth microinteractions
- Explanatory empty states
- Responsive grid layouts

## UX Writing Voice

Collaborative, simple, human. Avoid aggressive language.

**Use:** "Informar", "Relatar", "Acompanhar", "Apoiar"
**Avoid:** "Denunciar", "Reclamar", "Exigir", "Acusar"

**Example copy:**
- "Vamos cuidar da cidade juntos"
- "Sua informação ajuda a prefeitura a agir onde mais precisa"
- "O problema foi resolvido?"

## Accessibility

- AA contrast minimum (4.5:1)
- 14px minimum font size
- 44px minimum touch targets on mobile
- Don't rely on color alone for status
- Clear feedback after actions
